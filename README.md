# User Base Application

Веб‑приложение для управления пользователями.  
Реализована авторизация и базовый CRUD‑функционал: просмотр, создание, редактирование и удаление пользователей. Приложение разработано в рамках тестового задания.

## Основной функционал

- Авторизация пользователя  
- Просмотр списка пользователей  
- Создание пользователя  
- Редактирование пользователя  
- Удаление пользователя  
- Работа с REST API (mockapi.io)

## Технологический стек

- React  
- TypeScript  
- Ant Design  
- React Router v6  
- TanStack Query  
- Axios  
- styled-components  
- dayjs  

## Авторизация

Для входа в приложение используются моковые данные:

- Логин: `admin`  
- Пароль: `admin`  

## API

Для хранения и получения данных используется сервис [mockapi.io](https://mockapi.io).  

## Установка и запуск

### Установка зависимостей

npm install

### Запуск приложения

npm run start

Приложение будет доступно по адресу:
http://localhost:3000
